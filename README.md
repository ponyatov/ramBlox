#  `ramBlox`
## Dynamic RAM blocks Virtual Machine


* VM emulates vector CPU with RAM-block operations
* typed data structure interpretation on MCU devices
* visual debug/design (Python/Flask web interface)


### What do we need for GameDev SDK?

https://twitter.com/dponyatov/status/1392290368075571201?s=20

* custom language optimized for deep multitasking (Erlang-like?)
* virtual machine or native compiler with vector processor arch emulation
  (matrix, DSP, DMA-like blitz, message passing)
* distributed message passing + soft realtime green processes
